### cfc-schedules-tweet

Send tweets ahead of Chelsea FC's match!

### Feature(s)
1. Send a tweet ahead of Chelsea FC's match:
    - one day before the match
    - one hour before the match

### Example of the tweet
```Markdown
[MatchDay!]
#UCL 
🆚 Chelsea vs Real Madrid
🏟️ Stamford Bridge
📅 March 20, 2022
⏱️ 2AM GMT+7

#ChelseaFC #CFCFixture
```
